# Traders Market

**Traders Market** is a web application for selling and delivering MetaTrader 5 trading strategies. Visitors can learn about the product, sign up, and subscribers get access to strategy materials through a dashboard after purchase.

The app is built with **Next.js** and uses **Firebase** for authentication and user data, **Stripe** for payments, and supporting services (e.g. file storage and email) as configured in your environment.
